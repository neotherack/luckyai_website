Technologies

At this section we will cover all technologies used by the project.

Mongo DB

Many information is stored, classified, reviewed and analyzed to make decissions. BigData is one of the key technologies.

Ubuntu

Project started with Raspbian (Raspberry OS) but that was not powerful enough, so recently migrated project to Ubuntu server. Raspberry Pi 3 Model B, was taking more than five days to get converged networks, Ubuntu server takes only a few minutes/hours.

Python

This is one major tech for this project, being able to combine all others in one operating project. This programming language is flexible and powerful, plenty of resources and libraries. Most important lib is NeuroLab which is available at pip, its documentation is awesome.

Habilities

The project is just starting, so only a few features are working but many are coming.

Detect and store world wide Trending Topics

All this info is stored to be used later, this is the root process because allows AI to know which tweets are better for training

Retweet trending tweets

Randomized process will find most relevant tweets, it will decide when to retweet them.

Find best tweets for training

Not all training tweets are chosen from trendings, others are chosen randomly. This allows AI to better know which is trending and which one is not.

Next step: predict tweet impact on networks

Getting best training data will allow AI to guess the amount of retweets and favorites for one tweet, many information is analyzed to guess this info which is still experimental and not public.
